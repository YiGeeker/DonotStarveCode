\section*{输入方法}
游戏中按\framebox[1.5em][c]{\raisebox{-0.1em}{\parbox[b][1em][c]{0.5em}{\centering \raisebox{-1.8ex}{\texttt{\textasciitilde}}\\
                                                \raisebox{0ex}{\texttt{`}}}}}
\hspace{-0.3em}键开启控制台，输入代码后按\fbox{\rule[-0.1em]{0pt}{1em}\texttt{Enter}}
\hspace{-0.3em}键生效，按%
\fbox{\rule[-0.1em]{0pt}{1em}\texttt{Ctrl}}+\framebox[1.5em][c]{\rule[-0.1em]{0pt}{1em}\texttt{l}}
\hspace{-0.3em}键关闭。

\section*{代码格式}
\texttt{DebugSpawn('code')}，在鼠标停放的位置会出现一个想要的物品。例如：\texttt{DebugSpawn('resurrectionstone')}，在鼠标停放的位置会出现一个重生石。

\texttt{c\_spawn('code',number)}，在鼠标停放的位置会出现指定数量的想要物品。例如：\texttt{c\_spawn('tentaclespike',3)}，在鼠标停放的位置会出现三个狼牙棒。

\texttt{c\_give('code',number)}，直接在物品栏中出现指定数量的想要物品。例如：\texttt{c\_give('spidergland',3)}，在物品栏中出现三个蜘蛛腺体。
